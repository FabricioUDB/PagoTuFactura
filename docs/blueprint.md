# **App Name**: Invoice Hub

## Core Features:

- User Authentication: Secure user authentication using Firebase for login, registration, and logout.
- Invoice Creation: Allow users to create invoices with details like customer name, invoice number, items, and amounts.
- Invoice List: Display a list of invoices with status and key details.
- Invoice Detail View: Provide a detailed view of each invoice, including all line items and totals.
- PDF Generator: Use AI tool that can take structured invoice information and determine when the invoice is legally compliant

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and professionalism.
- Background color: Light gray (#F0F2F5), almost white, for a clean and modern look.
- Accent color: Amber (#FFC107) for call-to-action buttons and important information.
- Body and headline font: 'Inter', a sans-serif, for a modern and readable interface.
- Use a set of professional icons from a library like FontAwesome or Material Icons to represent different invoice actions and statuses.
- Clean and organized layout with clear sections for invoice creation, lists, and detail views. Use TailwindCSS grid system for responsiveness.
- Subtle transitions and animations to improve user experience, such as fading in new content or highlighting updated fields.
